Humble Light International School Website

The official website for Humble Light International School (HLIS), designed to provide students, parents, staff, and prospective families with easy access to important school information and resources.

Features

* School information and history
* Academic programs and curriculum details
* Admissions information
* News and announcements
* Contact and inquiry forms
* Responsive design for desktop and mobile devices

Technologies Used

* HTML5
* CSS3
* JavaScript
* WordPress
* Website Hosting & cPanel Management

Live Website

https://hlis.com.ng/

Purpose

This project serves as the school’s online presence, providing a professional platform for communication, information sharing, and engagement with the school community.
